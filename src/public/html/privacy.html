What data is collected?

Akkorokamui itself does not collect any data (as the source code shows), however:

  • if using a third-party instance, there may be some form of server-side logging (which is usually harmless); in this case, to get information, you must contact the instance administrator
  • if self-hosting, your IP address is shared with Firebaseio, Hnpwa, Hckrnews and Algolia in order to provide threads and searches

Cookies

Akkorokamui does not use cookies.
